# V11 - Hệ thống quản lý giao nhiệm vụ và đánh giá thực hiện
## Bản hoàn thiện triển khai chính thức cho Đảng ủy xã Ea Kiết

### Điểm nhấn V11
- Hoàn thiện triển khai LAN nội bộ + sẵn hướng mở internet an toàn
- Bộ tài liệu triển khai, vận hành, sao lưu, phục hồi
- Mobile app Android có đăng nhập, việc của tôi, cập nhật tiến độ, tải ảnh minh chứng
- Chuẩn hóa cấu hình để dùng lâu dài trong cơ quan

### Khởi động nhanh
1. Chạy install_requirements.bat
2. Chạy start_lan.bat
3. Truy cập http://IP_MAY_CHU:8000
